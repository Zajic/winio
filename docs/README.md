# Winio – Dokumentace a příprava (Blok 0)

Struktura výstupů přípravy projektu.

## Rozhodnutí (Blok 0.1, 0.2)

- **`rozhodnuti/odds-api-vyber.md`** – Výběr Odds API; kritéria, možnosti, tabulka rozhodnutí. Po dokončení vyplnit „Rozhodnutí“ a uložit API klíč do env.
- **`rozhodnuti/affiliate-programy.md`** – Checklist affiliate programů (sázkovky, kasina); odkazy na programy a pravidla. Citlivé affiliate URL ukládat až v DB aplikace.

## Právní texty (Blok 0.3)

- **`legal/obchodni-podminky.md`** – Návrh Obchodních podmínek (účel webu = pouze informace, žádné sázení u nás, 18+, financování, odpovědnost).
- **`legal/ochrana-osobnich-udaju.md`** – Návrh Zásad ochrany osobních údajů (GDPR): účely, právní základy, cookies, práva, kontakt.
- **`legal/cookie-listka.md`** – Text a požadavky pro cookie lištu (zobrazení při první návštěvě; odkaz na GDPR; Souhlas / Odmítnout).

V Bloku 3 se tyto texty použijí na stránkách `/obchodni-podminky`, `/ochrana-osobnich-udaju` a v komponentě cookie lišty. Před zveřejněním doplnit údaje o provozovateli a datum účinnosti; případně nechat právně zkontrolovat.
